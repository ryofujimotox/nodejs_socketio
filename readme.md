# Install

**install node.js**

``` bash
cd /〇〇/nodejs_socketio
npm install
```

**install SocketIO**

``` bash
npm install --save express socket.io
```

**Run**

``` bash
node index.js
```

**Confirm**

http://localhost:3000/
